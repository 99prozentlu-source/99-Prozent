# **App Name**: Club 99 SmartManager

## Core Features:

- Secure PIN-based Login: Members authenticate with a unique 4-digit PIN to access their respective user or admin interface.
- Member Dashboard & Balance: Each member views their current account balance, updated in real-time with transactions.
- Quick Item Booking: Members can rapidly book pre-defined items (e.g., beverages), automatically deducting costs from their balance.
- Admin Inventory Overview: Admins have access to a real-time list of all inventory items, highlighting those below minimum stock levels.
- AI-assisted Invoice Creation Tool: Admins can generate PDF invoices for customers, using a generative AI tool to assist in drafting item descriptions or common invoice details.
- Data Backup & Export: Admins can easily export all club data (members, inventory, transactions) as a JSON backup file.

## Style Guidelines:

- Color scheme: Dark mode, using a cool, vibrant primary green (HSL: 160, 80%, 50% / RGB: #1AD8AC) for prominence against a dark, subtly tinted background (HSL: 160, 15%, 10% / RGB: #16201D). An analogous brighter green accent (HSL: 130, 70%, 60% / RGB: #5ACF75) will be used for key interactive elements.
- Primary and secondary font: 'Inter' (sans-serif) for its modern, clean, and objective aesthetic, suitable for both headlines and body text within a data-centric application.
- Utilize clean, modern outline icons for navigation, actions, and status indicators. Emojis can be selectively integrated for representing quick-booking items for intuitive recognition.
- A responsive, mobile-first design focused on a centered, card-based content display. The layout emphasizes clear data presentation, large, touch-friendly buttons, and intuitive navigation for both user and admin functionalities.
- Subtle and quick transitions for screen changes and states (e.g., button presses, item booking confirmations) to provide immediate user feedback without delaying interactions.
